# IBM Plex Sans Thai Font Installation

Please download the following font files from Google Fonts and place them in the `assets/fonts/` directory:

1. **IBMPlexSansThai-Regular.ttf**
   - URL: https://github.com/google/fonts/raw/main/ofl/ibmplexsansthai/IBMPlexSansThai-Regular.ttf

2. **IBMPlexSansThai-SemiBold.ttf**
   - URL: https://github.com/google/fonts/raw/main/ofl/ibmplexsansthai/IBMPlexSansThai-SemiBold.ttf

3. **IBMPlexSansThai-Bold.ttf**
   - URL: https://github.com/google/fonts/raw/main/ofl/ibmplexsansthai/IBMPlexSansThai-Bold.ttf

## Steps:
1. Create the directory: `assets/fonts/` (if it doesn't exist)
2. Download each font file from the URLs above
3. Place the downloaded `.ttf` files in `assets/fonts/`
4. Restart the Expo development server

The app will automatically load these fonts when it starts.
